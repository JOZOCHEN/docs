git国内下载站 <https://github.com/waylau/git-for-win>

生成SSH-KEY: ssh-keygen -C "email"

## 指令  
配置用户名/邮箱：  
git config --global user.name "nameVal"  
git config --global user.email "eamil@qq.com"  

读取用户名/邮箱：  
git config user.name  
git config user.email
